# One Day at a Time 🌿

A personal recovery website built with love for the journey of healing, growth, and gratitude.

## Features

### 🏠 Home
- Clean time calculator - track your recovery milestone
- Welcoming introduction to your recovery space
- Quick access to daily gratitude and meditation

### 📖 Just for Today
- Daily link to NA World Services "Just for Today" meditation
- Beautiful, calming design for daily reflection

### 🙏 Gratitude List
- Write 5 things you're grateful for each day
- Save entries to your browser's local storage
- **Share with fellows** - generate a unique link to share your gratitude list with your recovery community
- View past entries with date stamps

### 📝 Journal
- Personal journaling space for reflection
- Mood tracking with emoji indicators
- Private entries stored locally

### 📍 Resources
- Links to CMA meetings (including Boston area)
- CMA literature and resources
- Just for Today daily meditations
- SAMHSA helpline information

## 🚀 Setup on GitHub Pages

1. **Create a new repository** on GitHub named `recovery-site` (or whatever you prefer)

2. **Upload the files:**
   - Go to your new repository
   - Click "uploading an existing file"
   - Drag and drop the `index.html` file

3. **Enable GitHub Pages:**
   - Go to repository Settings
   - Scroll down to "Pages" in the left sidebar
   - Under "Source", select "Deploy from a branch"
   - Select "main" branch and "/ (root)" folder
   - Click Save

4. **Access your site:**
   - Your site will be live at: `https://k-jayp.github.io/recovery-site/`
   - It may take a few minutes to deploy

## 💾 Data Storage

All your data (clean date, gratitude entries, journal entries) is stored in your browser's local storage. This means:
- ✅ Your data stays private on your device
- ✅ No account needed
- ⚠️ Data is specific to each browser/device
- ⚠️ Clearing browser data will remove entries

## 🔗 Sharing Gratitude

When you save a gratitude entry, you can generate a shareable link. This link encodes your gratitude items directly in the URL, so:
- Fellows can view your gratitude list by clicking the link
- No server or database needed
- The shared view encourages them to write their own!

## 🎨 Customization

Feel free to customize the site! Some ideas:
- Change the color scheme in the CSS `:root` variables
- Add your own quotes to the footer
- Modify the resources to include your specific local meetings
- Add more pages as needed
